# Demo Environment Configuration

# For local development - create a .env file with:
# VITE_GEMINI_API_KEY=your_demo_api_key_here

# For GitHub Pages deployment:
# 1. Create restricted API key at: https://makersuite.google.com/app/apikey
# 2. Set application restrictions to your GitHub Pages URL
# 3. Add to GitHub Secrets as: GEMINI_API_KEY
# 4. Deploy via GitHub Actions

# Recommended API restrictions for demo:
# - APIs: Generative Language API only
# - Referrers: *.github.io/*, yourusername.github.io/NexoraHackathon/*
# - Quota: Set reasonable daily limits
